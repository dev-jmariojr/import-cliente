# App Importação de Arquivo
Aplicação para importação de arquivo texto referente ao cadastro de clientes e operações de cadastro (inserção, consulta, atualização e exclusão).

```Teste para uma vaga de programador delphi em 2006```

Estava repassando a carreira e me lembrei de um teste que fiz para uma vaga de programador delphi em 2006. 
O teste consistia em duas etapas:
- Fazer a importação do arquivo para uma base SQL Server.
- Fazer uma tela de cadastro, com as operações de inserção, consulta, atualização e exclusão.
(Nesta parte, tinha restrição quanto aos componentes da paleta ```Data Control``` apenas o DBGrid poderia ser utilizado, para apresentar os dados importados)

Na ocasião, era a minha primeira experiência profissional como programador, só havia feito estágio. A linguagem era Delphi. Já tinha conhecimento básico de SQL e consegui desenrolar.

Não fiz a primeira parte do teste, apenas a segunda, mas acabei sendo contratado e durante 8 anos e meio trabalhei nesta empresa, 
podendo me desenvolver como profissional e adquirir bastante conhecimento. Foi a minha primeira experiência profissional como programador e a mais longa.

Dai resolvi refazer, com algumas adaptações. Gerei um arquivo mais ou menos como era o layout, baixei a versão free do Delphi 12 (Embarcadeiro) e resolvi usar um banco de dados mais simples, o SQLite, para não ter que me preocupar com a configuração do SQL Server.

Até que me lembrei de muita coisa de delphi, para quem não utilizava delphi desde 2021.
